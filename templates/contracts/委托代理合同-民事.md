# 委托代理合同-民事 字段清单

## 模板文件

- 模板路径：`/Users/clukay/Program/feishu-opencode-bridge/templates/contracts/委托代理合同-民事.docx`
- 模板类型：民事委托代理合同
- 推荐场景：劳动争议、一般民商事纠纷、执行代理

## 拆分原则

- 固定不动：律所信息、委托人须知、通用义务条款、保密条款、通知送达、风险告知书
- 必填字段：合同首页和案件识别所需信息
- 可选字段：决定程序范围、收费模式、争议解决、特别约定
- 条件字段：只有在选中特定收费模式或程序时才展示
- 受限字段：风险告知书是否输出由收费模式决定

## 第一层：基础必填字段

### 合同头部

- `contract_no`
  说明：合同编号
  类型：文本
  默认：可留空，后续支持自动生成

- `sign_date`
  说明：签约日期
  类型：日期

- `sign_city`
  说明：签约地点
  类型：文本
  默认：深圳

### 甲方信息

- `client_name`
  说明：甲方名称
  类型：文本

- `client_representative`
  说明：法定代表人/负责人/授权代表
  类型：文本

- `client_id_code`
  说明：身份证号/统一社会信用代码
  类型：文本

- `client_address`
  说明：甲方地址
  类型：多行文本

- `client_email`
  说明：电子邮箱
  类型：文本

- `client_phone`
  说明：联系电话
  类型：文本

### 案件信息

- `counterparty_name`
  说明：对方当事人名称
  类型：文本

- `case_cause`
  说明：案由
  类型：文本

- `lead_lawyer`
  说明：承办律师
  类型：文本

## 第二层：核心选项

### 委托程序

- `engage_arbitration`
  说明：是否委托仲裁阶段
  类型：布尔

- `engage_first_instance`
  说明：是否委托一审诉讼
  类型：布尔

- `engage_second_instance`
  说明：是否委托二审诉讼
  类型：布尔

- `engage_enforcement`
  说明：是否委托执行程序
  类型：布尔

- `engage_settlement`
  说明：是否包含调解/和解事宜
  类型：布尔

### 授权方式

- `authorization_type`
  说明：授权类型
  类型：单选
  取值：
  - `general`
  - `special`

- `special_authorization_scope`
  说明：特别授权事项
  类型：多行文本
  条件：`authorization_type = special`

### 收费模式

- `fee_mode`
  说明：收费模式
  类型：单选
  取值：
  - `stage_fixed`
  - `base_plus_risk`
  默认：`stage_fixed`

### 办案费用承担方式

- `expense_mode`
  说明：办案费用承担方式
  类型：单选
  取值：
  - `lump_sum`
  - `reimbursement`

### 争议解决方式

- `dispute_resolution_mode`
  说明：争议解决方式
  类型：单选
  取值：
  - `szia_arbitration`
  - `court_litigation`
  - `custom`

- `dispute_resolution_text`
  说明：争议解决条款自定义文本
  类型：多行文本
  条件：`dispute_resolution_mode = custom`

## 第三层：条件字段

### 按阶段收费

- `fee_arbitration`
  说明：仲裁阶段律师费
  类型：金额
  条件：`fee_mode = stage_fixed` 且 `engage_arbitration = true`

- `fee_first_instance`
  说明：一审阶段律师费
  类型：金额
  条件：`fee_mode = stage_fixed` 且 `engage_first_instance = true`

- `fee_second_instance`
  说明：二审阶段律师费
  类型：金额
  条件：`fee_mode = stage_fixed` 且 `engage_second_instance = true`

- `fee_enforcement`
  说明：执行阶段律师费
  类型：金额
  条件：`fee_mode = stage_fixed` 且 `engage_enforcement = true`

### 基础收费 + 风险收费

- `base_fee`
  说明：基础费用
  类型：金额
  条件：`fee_mode = base_plus_risk`

- `risk_fee_rate`
  说明：风险收费比例
  类型：百分比
  条件：`fee_mode = base_plus_risk`

- `risk_fee_basis`
  说明：风险收费计算基准
  类型：单选
  取值：
  - `recovered_amount`
  - `reduced_debt`
  - `custom`
  条件：`fee_mode = base_plus_risk`

- `risk_fee_basis_custom_text`
  说明：风险收费计算基准自定义说明
  类型：多行文本
  条件：`fee_mode = base_plus_risk` 且 `risk_fee_basis = custom`

### 付款要求

- `payment_due_days`
  说明：签约后付款期限
  类型：数字
  默认：3

- `late_fee_daily_rate`
  说明：逾期付款违约金日比例
  类型：文本
  默认：日万分之*

- `late_fee_terminate_days`
  说明：逾期多少日乙方可解除合同
  类型：数字
  默认：15

## 第四层：可删减条款

这些不建议让用户逐字编辑，而建议做成“保留/删除”开关。

- `keep_risk_notice_attachment`
  说明：是否保留《风险代理告知书》
  类型：布尔
  默认：
  - `fee_mode = base_plus_risk` 时强制保留
  - `fee_mode = stage_fixed` 时默认删除

- `keep_indirect_conflict_waiver`
  说明：是否保留“间接利益冲突豁免”条款
  类型：布尔
  默认：保留

- `keep_property_preservation_note`
  说明：是否保留“财产保全另行委托”条款
  类型：布尔
  默认：按案件是否涉及保全决定

- `keep_co_counsel_clause`
  说明：是否保留“乙方可与其他律师或律所合作办理”条款
  类型：布尔
  默认：保留

## 第五层：补充字段

- `special_terms`
  说明：特别约定
  类型：多行文本

- `client_contact_person`
  说明：甲方指定联系人
  类型：文本

- `invoice_required`
  说明：是否要求开票
  类型：布尔
  默认：是

- `invoice_title`
  说明：发票抬头
  类型：文本
  条件：`invoice_required = true`

## 风险告知书规则

- `fee_mode = stage_fixed` 时，不输出《风险代理告知书》
- `fee_mode = base_plus_risk` 时，自动附带《风险代理告知书》
