# 4/20 Memory 模块范围说明

## 结论

当前仓库 **没有** memory 子系统实现。4/20 提交版不包含以下能力：

- memory store / recall
- embedding 检索
- vector 存储
- Obsidian 同步
- `profile.md` 输出
- fact extraction

这些条目在本仓库里应视为 **out of scope**，需要单独的 memory 子系统设计与实现，不属于本次提交流程的阻塞项。

## 检查依据

本次已对仓库关键位置做检索：

- `src/`
- `package.json`
- `docs/`

使用的关键词包括：

- `embedding`
- `memory`
- `vector`
- `obsidian`
- `profile.md`
- `recall`
- `fact extraction`

结果显示：

- `src/` 下没有 `memory/` 模块或等价实现
- 运行时代码没有 recall 注入、事实提取、向量检索、Obsidian 输出路径
- 依赖中没有与向量存储或 memory 管线直接对应的运行时包

## 对 4/20 验收的影响

本次提交版应把 memory 相关检查项归类为：

- 文档说明：已明确超出当前仓库范围
- 验收结论：不计入本次功能缺陷
- 后续工作：作为独立功能线单独规划
